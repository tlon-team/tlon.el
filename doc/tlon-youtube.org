* YouTube Functionality (=tlon-youtube.el=)
:PROPERTIES:
:CUSTOM_ID: h:tlon-youtube
:END:

This module provides functionality to process and upload podcast episodes to YouTube.

** Menu
:PROPERTIES:
:CUSTOM_ID: h:tlon-youtube-menu
:END:

#+findex: tlon-youtube-menu
+ ~tlon-youtube-menu~ :: Opens the main transient menu for YouTube-related operations.

** Video Generation
:PROPERTIES:
:CUSTOM_ID: h:tlon-youtube-video-generation
:END:

#+findex: tlon-youtube-generate-wavelength-video
+ ~tlon-youtube-generate-wavelength-video~ :: Generates a 720p video (1280x720 pixels) with an animated wavelength from an audio file using ~seewav~. Prompts the user to select an audio file from the "uqbar-audio" repository. The output video is saved in the ~paths-dir-downloads~ directory with a ~.mp4~ extension, using the original audio file name. This command can also be accessed via the [[h:tlon-youtube-menu][tlon-youtube-menu]].
