This manual describes the features and customization options for the Emacs Lisp file =tlon-search.el=.

* Overview
:PROPERTIES:
:CUSTOM_ID: h:overview
:END:

=tlon-search= provides search commands for the Tlon workflow, integrating with =magit= for Git history searches, =consult= (via =ripgrep=) for file content searches, =forge= for issue searches, and external web services for translation lookups. All commands are accessible through a unified transient menu.

:CUSTOM_ID: h:commands

** Git history searches
:PROPERTIES:
:CUSTOM_ID: h:git-history-searches
:END:

*** ~tlon-search-commits~
:PROPERTIES:
:CUSTOM_ID: h:tlon-search-commits
:END:
#+findex: tlon-search-commits

Search for SEARCH-STRING in REPO's commit history.

When called interactively, you are prompted for the search string. If REPO is =nil=, the current directory is used. This command calls ~magit-log-all~ with the =--grep= argument, displaying all commits whose messages match the search string in a Magit log buffer.

*** ~tlon-search-commit-diffs~
:PROPERTIES:
:CUSTOM_ID: h:tlon-search-commit-diffs
:END:
#+findex: tlon-search-commit-diffs

Search for SEARCH-STRING in REPO's commit diff history.

When called interactively, you are prompted for the search string. If REPO is =nil=, the current directory is used. This command calls ~magit-log-all~ with the =-S= (pickaxe) argument, displaying commits where the string was added or removed in the diff content. This is useful for finding when a specific piece of code or text was introduced or removed.

** File content search
:PROPERTIES:
:CUSTOM_ID: h:file-content-search
:END:

*** ~tlon-search-files~
:PROPERTIES:
:CUSTOM_ID: h:tlon-search-files
:END:
#+findex: tlon-search-files

Search for SEARCH-STRING in REPO files using =consult-ripgrep=.

When called interactively, you are prompted for the search string. If REPO is =nil=, the current repository is determined using ~tlon-get-repo~ (with the =include-all= flag). Results are presented in the minibuffer with live preview via =consult-ripgrep=.

** Translation search
:PROPERTIES:
:CUSTOM_ID: h:translation-search
:END:

*** ~tlon-search-for-translation~
:PROPERTIES:
:CUSTOM_ID: h:tlon-search-for-translation
:END:
#+findex: tlon-search-for-translation

Search for a Spanish translation of an English STRING using multiple online resources and a local dictionary.

When called interactively, you are prompted for the English string to translate. The command opens the following websites in new browser tabs, each searching for the provided string:

- Spanish Stack Exchange
- Bab.la (English-Spanish dictionary)
- Wikipedia (English, full-text search)
- Reverso Context (English-Spanish translation)
- Linguee (English-Spanish search)

It also searches for the string in the locally installed GoldenDict-NG application via ~goldendict-ng-search-string~.

** Transient menu
:PROPERTIES:
:CUSTOM_ID: h:transient-menu
:END:

*** ~tlon-search-menu~
:PROPERTIES:
:CUSTOM_ID: h:tlon-search-menu-command
:END:
#+findex: tlon-search-menu

Display a =transient= menu interface for accessing search commands. The menu provides the following options under the "Search" group:

| Key | Description              | Command                          |
|-----+--------------------------+----------------------------------|
| =c= | Commits                  | ~tlon-search-commits~            |
| =d= | Commit diffs             | ~tlon-search-commit-diffs~       |
| =f= | Files                    | ~tlon-search-files~              |
| =i= | Issues (current repo)    | ~forge-search~                   |
| =I= | Issues (all repos)       | ~tlon-forge-search~              |
| =t= | Translation              | ~tlon-search-for-translation~    |
